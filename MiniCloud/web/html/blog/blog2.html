Tại sao tôi lại chọn Docker Secrets thay vì .env?

Ngôn Dev • May 20, 2024 • 10 min read

Trong dự án MiniCloud lần này, tui đã "chốt hạ" sử dụng **Docker Secrets**. Tại sao ư? Hãy nhìn vào cách nó bảo vệ data của ông Việt (Backend).

⚠️ Vấn đề của .env:

Nó nằm trần trụi dưới dạng file text. Nếu ai đó có quyền truy cập container, họ thấy hết. Docker Secrets thì khác, nó được mount dưới dạng file ảo trong RAM tại `/run/secrets/`.

Code mẫu cho anh em:


# docker-compose.yml
services:
  app:
    secrets:
      - db_password
secrets:
  db_password:
    file: ./secrets/db_pass.txt

Kết quả là ông Việt chỉ cần đọc file này trong Python cực kỳ an toàn. Hệ thống vững như bàn thạch!
